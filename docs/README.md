# README.md

## Sistema de Gestão de Passeios

Sistema web interno para gerenciamento de vendas de passeios turísticos do hotel.

### Objetivos

* Substituir o controle atual realizado em planilhas.
* Centralizar informações de vendas de passeios.
* Controlar passageiros, embarcações e comissões.
* Armazenar comprovantes de pagamento.
* Gerar relatórios operacionais e financeiros.

### Stack Tecnológica

#### Backend

* Python
* FastAPI
* SQLAlchemy
* PostgreSQL

#### Frontend

* HTML5
* CSS3
* JavaScript
* Bootstrap

#### Infraestrutura

* Docker
* Caddy

### Funcionalidades Principais

* Login de usuários
* Cadastro de vendas
* Cadastro de passeios
* Cadastro de embarcações
* Upload de comprovantes
* Controle de status das vendas
* Dashboard gerencial
* Relatórios de vendas
* Relatórios de comissões
