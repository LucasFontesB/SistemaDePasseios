# VISION.md

## Visão do Produto

O sistema tem como objetivo facilitar o gerenciamento dos passeios turísticos vendidos pelo hotel.

Atualmente as informações são registradas em planilhas, o que dificulta consultas, relatórios e controle operacional.

O sistema deverá permitir que recepcionistas e gestores acompanhem todo o ciclo de venda de um passeio.

## Público-Alvo

### Recepcionistas

Responsáveis por:

* Registrar vendas
* Consultar reservas
* Atualizar status
* Anexar comprovantes

### Gerência

Responsável por:

* Acompanhar indicadores
* Consultar relatórios
* Conferir comissões
* Avaliar desempenho das vendas

### Administração

Responsável por:

* Gerenciar usuários
* Configurar passeios
* Configurar embarcações
* Acessar todas as informações

## Indicadores de Sucesso

* Redução de uso de planilhas
* Consulta rápida de vendas
* Controle centralizado de comprovantes
* Relatórios automáticos
* Controle de comissões sem cálculos manuais
